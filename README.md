# Research Skills

A growing collection of reusable workflows for scientific research.

## Skills

| Skill | Purpose |
|---|---|
| Paper Reading | Understand scientific papers systematically |
| Scientific Writing | Organize manuscript writing and academic expression |
| Figure Analysis | Learn how to read and explain scientific figures |
| Experiment Planning | Convert research questions into executable experiments |
| Reviewer Response | Organize point-by-point responses to reviewers |
| Literature to Project | Turn paper reading into research ideas |
